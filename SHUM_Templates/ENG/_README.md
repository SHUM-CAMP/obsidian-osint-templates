# ENG Report Pack

- Use `SHUM_Templates/ENG/MD/` when authoring inside Obsidian.
- Use `ReportTemplates/ENG/DOCX/` for external delivery workflows.
- Source snapshots remain in `SHUM_Templates/ENG/MD/` and `SHUM/ENG/RTF/`.
- For in-app DOCX reading, open files directly in Obsidian (`docx-viewer` default).
- If needed, fallback to `docxer` or open externally with `open-with`.
