# Analyst Levels Checklist

## Junior
- [ ] Uses metadata contract correctly
- [ ] Produces linked source/evidence notes
- [ ] Completes QA checklist with guidance

## Intermediate
- [ ] Runs case lifecycle independently
- [ ] Applies at least 3 playbooks correctly
- [ ] Produces bilingual-ready report drafts

## Senior
- [ ] Leads review queue
- [ ] Maintains methodology playbooks
- [ ] Mentors onboarding analysts
