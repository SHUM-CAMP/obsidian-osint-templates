# Role Dashboards

## Lead dashboard
- Active cases by status
- Stale cases (`last_reviewed` older than threshold)
- Queue load by assignee

## Analyst dashboard
- Assigned cases
- Today's next actions
- Missing metadata warnings

## Reviewer dashboard
- Cases in `review`
- QA checklist completion
- Reports awaiting signoff

## Dataview-ready fields
- `case_id`
- `status`
- `assignee`
- `priority`
- `last_reviewed`
