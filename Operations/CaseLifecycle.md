# Case Lifecycle

## Workflow
1. Intake
2. Assignment
3. Investigation
4. Review
5. Publication

## Entry criteria by stage
- Intake: case note created with metadata contract fields.
- Assignment: owner and deadline set.
- Investigation: evidence and source notes linked.
- Review: QA checklist completed, reviewer assigned.
- Publication: final report approved and archived.

## Exit criteria
- A case can be closed only after publication status and final review timestamp are set.
