# OSINT Vault Home

This repository is a single, clone-ready Obsidian vault that combines:
- WebBreacher original templates and SOP materials
- SHUM methodology and reporting templates (ENG + UKR)

## Start Here
- Open `01_QuickStart.md`
- Browse `Templates/_index.md`
- Browse `Methodology/_index.md`
- Browse `ReportTemplates/_index.md`
- Browse `Examples/_index.md`
- Browse `Operations/_index.md`
- Browse `Reports/_index.md`
- Browse `Training/_index.md`
- Validate coverage with `Templates/_coverage_checklist.md`
- Validate report packs with `ReportTemplates/_coverage_checklist.md`

## Source Attribution
- WebBreacher upstream vault and materials: `WebBreacher originals/`
- SHUM-specific templates and docs: `SHUM/`

Both sources are preserved side-by-side and referenced through curated index pages.
