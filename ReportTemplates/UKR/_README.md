# UKR Report Pack

- Use `ReportTemplates/UKR/MD/` when authoring inside Obsidian.
- Use `ReportTemplates/UKR/DOCX/` for external delivery workflows.
- Source snapshots remain in `SHUM/UKR/MD/` and `SHUM/UKR/RTF/`.
- For in-app DOCX reading, open files directly in Obsidian (`docx-viewer` default).
- If needed, fallback to `docxer` or open externally with `open-with`.
- Included Markdown templates:
  - `Анкета персональних даних.md`
  - `Звіт щодо організації.md`
  - `Інструкція по заповненню робочих шаблонів.md`
  - `Контроль уражень.md`
  - `Локація.md`
  - `Методологія та шаблони.md`
- Included DOCX templates:
  - `Шаблон Анкета персональних даних.docx`
  - `ШаблонОрганізації.docx`
  - `Інструкція по заповненню робочих шаблонів.docx`
  - `ШаблонКУ.docx`
  - `Шаблон Локація.docx`
  - `Методологія та шаблони.docx`
