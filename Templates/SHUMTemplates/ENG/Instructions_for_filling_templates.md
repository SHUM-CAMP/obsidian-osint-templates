# Instructions for Filling Working Templates

## Personal Data Profile

- The base template must not be altered or edited manually.
- Create one copy of the template per person in the `Profiles` folder.
- Rename each file to the target person's full name.
- Do not modify labels in the left column.
- Use Times New Roman: full name fields (RU + UKR) at size 16; all other fields at size 12.
- Fill the document in Ukrainian, except name fields or direct quotations.
- In multi-value lists, separate entries with `;`.
- Use date format `DD.MM.YYYY`.
- Translate addresses to Ukrainian when possible; add a comment if untranslated.
- Keep workplaces and organizations concise; move expanded biography notes to the notes section.
- Enter phone numbers, IDs, and document numbers without spaces or hyphens.
- Use full international phone format (e.g., `+7`, `+38`).
- Verify all links; they must be active.
- Select the proper relationship type for related persons.
- If there are multiple related persons, duplicate the related-person section before notes.
- Put clarifications into the comments column.
- Add useful extra information to notes if no dedicated field exists.

## Location

- The base template must not be altered or edited manually.
- Copy the template to your working folder before filling.
- Create a separate file for each location in the `Locations` folder.
- Rename the file to the location name.
- Fill the document in Times New Roman, size 12.
- Main language is Ukrainian.
- Coordinates must be in WGS-84.
- Address format: region, city, street, building, apartment.
- Contact phone numbers should be in `+7` or `+38` format.
- Leadership format: full name; position.
- Add a satellite screenshot (Google Earth or equivalent) with the building clearly marked.
- Add current building photos, ideally from multiple angles.
- Put any additional important details into notes.

## Government Authorities Guide

- Each folder should contain a regional copy used to create profiles by template.
- Before creating a profile, verify each position and full name for current relevance.
- If current data differs from the guide file, add a comment in your working document.
- Create profiles only for persons with enough available information.
